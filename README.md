# Nervape Bridge Backend

### Install

```sh
$ yarn install
```

### Build

```sh
$ yarn build
```

### Start in development

```sh
$ yarn start:detector
```

```sh
$ yarn start:parser
```

```sh
$ yarn start:minter
```

### Start in production


```sh
$ yarn start:prod:detector
```

```sh
$ yarn start:prod:parser
```

```sh
$ yarn start:prod:minter
```
